# 本科就业概述

### 以下是 21 级老登留下的一段经验感受

我就说一下，多数找工作的同学都会面临的，校招和市场化就业，校招呢，相对于社招来说还有一定的新手保护期。八股项目算法三件套，其实，先人们都已经总结出很多经验和案例了，但是对于没什么面试经验的同学来说，都会有一个成长的过程（不论你是找实习还是正式工作。）越早开始，越早准备越好，在经历过足够多的面试之后，你也就知道自己的优势和长处了，对自己项目的细节以及可能被问到的问题也是心中有数了，如果你还没到找工作的时候，那么就多做好积累吧，让自己的简历上有足够的内容可以用来推销自己，毕竟，巧妇难为无米之炊，面试技巧掌握的再好，如果简历太空洞，那也不会有啥面试机会，祝大家都找到心仪的工作。

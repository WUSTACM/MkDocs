# Coding 俱乐部介绍

## 一、社团简介

Coding 俱乐部是武汉科技大学计算机学院极具特色的社团，作为学院首个且唯一的五星级社团，在校园编程文化传播与人才培养中扮演重要角色。这里汇聚对编程热爱、渴望成长的同学，以“发现热爱，享受青春”为理念，为成员打造学习、实践、交流的优质平台 。

---

## 二、招新部门及职责

### （一）后勤部

锻炼成员统筹协调能力，负责社团物资管理、活动场地安排、流程规划等工作。从活动筹备到执行，需细致规划、高效执行，助力成员收获可靠处事能力，保障社团各类活动有序开展 。

### （二）新媒体部

聚焦文案创作、设计与运营思维培养。负责社团线上平台（如公众号、抖音等）内容策划、推送，通过创意文案、精美设计，解锁线上传播技巧，提升对内容和用户的敏锐度，打造社团对外宣传窗口 。

### （三）程序部

深化编程技术为核心，适合想提升代码能力的同学。成员将在项目实践中，培养逻辑思维与问题解决能力，钻研 Scratch 等编程技术，制作游戏等作品，让创意落地，积累丰富项目经验 。

### （四）宣传部

围绕创意策划与沟通表达展开。负责活动海报设计、宣传策划、推广传播，通过多元方式传递社团信息，提升社团知名度，同时助力成员收获审美与执行力的双重提升 。

---

## 三、社团优势与特色活动

### （一）技能培养

加入社团，能接触 Scratch 编程等技术，学长学姐“手把手”教学，从代码钻研到游戏制作，让创意在指尖绽放，快速成长为编程小达人，为编程学习与实践筑牢基础 。

### （二）竞赛支持

为有竞赛需求的成员提供全方位服务。不仅及时推送竞赛信息，还传承学长学姐备赛经验，从入门指导到赛前冲刺，助力成员在编程竞赛中崭露头角，斩获佳绩 。

### （三）志愿活动

每周开展志愿活动，契合学校毕业需 30h 志愿工时要求。成员参与活动，既能充实校园生活，又能通过外出讲课等实践，拓展思维、赚取报酬，实现个人成长与志愿奉献双赢 。

### （四）宣讲会等交流活动

定期举办线上线下宣讲会，介绍社团起源、发展历程与精彩瞬间，还有管理层快闪亮相，展现社团活力与实力。成员可在交流中，深入了解社团文化，结识志同道合伙伴，融入编程大家庭 。

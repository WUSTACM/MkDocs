# 该如何转入计算机学院

转入计算机学院要参加一场上机考试和一场线下面试。下面是来自 24 级成功转入的学长的经验帖

### 准备上机考试：

因为我 24 年转专业是参加 C 语言 PAT 考试，无法提交 C++代码，不知道以后会不会支持其他语言，总之先务必在[考试官网](https://pintia.cn/problem-sets/994805046380707840/exam/problems/type/7)
练习 C 语言刷前一百道题。
如果还有时间可以刷武科大官网的题[wustoj](http://10.99.16.17/contest/322/problems)注册个账号，在 contest 里面随便定一个编号 2 开头的教学班，做这个班发的每一章的题，密码就是班的四位编号。

### C 语言教程：

推荐 C 语言跟着 B 站比特鹏哥网课 BV1Vm4y1r7jY 学习，主要是看 124p 前视频，跟着做视频里的练习。我刷题的时候发现用不到递归，指针，视频后面的项目井字棋，扫雷也可以不看。
一开始刷题语法错误，运行超时是常有的，有什么问题的时候可以去问 AI 或者 B 站，CSDN 搜题解，不必因为别人说这题简单太水而灰心，初学者就是慢慢积累经验。

上机考试时只能用机房电脑上的编译器：
虽然 B 站教程用的编译器很先进但上机考试却没有，我当时不熟悉考试机房的编译器 Dev-c++和 Code::block 考场上浪费了很多时间。现在我搜到了 Dev-c++的安装包和运行调试教程给大家参考
[安装包](https://pan.baidu.com/s/1I7Q5Y1u7UbW1aERDG6v1PQ)提取码：vytx
[运行调试教程](https://v.douyin.com/f93idE22meI)
[Dev-c++安装教程](https://www.bilibili.com/opus/1016751662433304595?from=search&spm_id_from=333.337.0.0)
[和初步使用教程](https://www.bilibili.com/opus/1004738308214882329?spm_id_from=333.1387.0.0)

### 在"网上办事大厅"报名参加转专业：

离考试两星期时老师会转发本科生院的转专业的通知链接，24 年的如下[24 年报名网站](http://jwc.wust.edu.cn/info/1371/44092.htm)按照此网站通知在"网上办事大厅"填写个人信息申请转入学院和专业
不久后计算机学院就会发送本学院转专业考试通知[通知网址](https://jsjkx.wust.edu.cn/info/1401/38882.htm)，要加里面的转专业通知 QQ 群，群里会发上机考试通知地点和时间。另外这个网站其实提到了通过考试后续面试的事"请同学们关注手机电话短信通知群学院网站等信息，勿错过重要信息！参加面试的同学面试时请携带一份个人情况简介及一份武汉科技大学成绩单！"

### 感觉面试没有上机考试重要：

面试其实是辅导员和两名老师问些简单的问题，比如在原来专业学了有什么计算机相关的课，用到了什么计算机软件，有什么常见的计算机语言。
个人情况简介我就随便写了下自我介绍和个人学习情况就打印出来了。
武汉科技大学成绩单是在教十一楼的服务中心用学生证打印的，当时其他考试还没有出成绩，所以打印出来其实只有军训成绩，总之交上就可以了。

所以主要是要准备上机考试，这么多天下来还是考验耐心，坚持，遇到难题没信心了可以去写之前的简单题找回自信然后问 AI 看 B 站，CSDN 里的题解，坚持刷完那些题，及时复习。
